FAQ

Questions fréquentes

📧 Comment fonctionne l'alias email ?

KeepID génère un alias @keepid.me unique pour chaque site directement dans le formulaire. Les emails sont redirigés vers votre vraie boîte. Votre vraie adresse n'est jamais exposée.

🔐 KeepID peut-il lire mes mots de passe ?

Non. Vos données sont chiffrées avec AES-256-GCM sur votre appareil. Même nous ne pouvons pas les lire — c'est le Zero-Knowledge.

🔥 Que se passe-t-il si je "Burn" un alias ?

L'alias est définitivement désactivé. Les spams s'arrêtent instantanément. Votre vraie adresse n'est pas affectée.

📥 Comment importer mes mots de passe existants ?

Compatible avec 1Password (CSV), Bitwarden (JSON), Trousseau Apple (CSV), Chrome, Safari et Firefox. Paramètres → Importer mes identifiants.

🔑 J'ai perdu mon mot de passe maître

Utilisez votre kit de récupération d'urgence. Sans ce code, l'accès est définitivement perdu — notez-le sur papier.

💳 Le plan gratuit est-il vraiment gratuit ?

Oui, sans limite de durée. 3 alias @keepid.me, 3 identités, générateur de mots de passe et vérification HIBP inclus. Aucune carte bancaire requise.
